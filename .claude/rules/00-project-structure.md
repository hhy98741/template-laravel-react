# Project Structure

This document describes the directory structure and file organization specific to this codebase.

## Backend Structure

```
app/
├── Actions/
│   └── Fortify/          # Auth actions (CreateNewUser, ResetUserPassword, etc.)
├── Concerns/             # Reusable validation traits
├── Http/
│   ├── Controllers/
│   │   └── Settings/     # Settings domain controllers
│   ├── Middleware/       # Custom middleware (HandleAppearance, HandleInertiaRequests)
│   └── Requests/
│       └── Settings/     # Form Request classes for settings
├── Models/               # Eloquent models
└── Providers/            # Service providers (AppServiceProvider, FortifyServiceProvider)
```

## Frontend Structure

```
resources/js/
├── actions/              # Wayfinder-generated controller action functions (auto-generated)
├── routes/               # Wayfinder-generated named route functions (auto-generated)
├── components/
│   ├── ui/              # Base UI components (Button, Input, Dialog, etc.)
│   └── *.tsx            # Feature components (app-shell, nav, breadcrumbs, etc.)
├── hooks/               # Custom React hooks
├── layouts/
│   ├── app-layout.tsx   # Main authenticated layout
│   ├── auth-layout.tsx  # Public/auth pages layout
│   ├── app/             # App layout sub-components
│   └── auth/            # Auth layout sub-components
├── lib/                 # Utility functions (utils.ts with cn() helper)
├── pages/               # Inertia page components
│   ├── auth/            # Authentication pages
│   ├── settings/        # Settings pages
│   └── *.tsx            # Top-level pages (dashboard, welcome)
└── types/               # TypeScript type definitions
```

## Route Organization

```
routes/
├── web.php              # Main web routes (includes settings.php)
├── settings.php         # Settings-specific routes (profile, password, appearance, etc.)
└── console.php          # Artisan console commands
```

## Configuration

- `bootstrap/app.php` - Middleware, exceptions, routing configuration (Laravel 12 style)
- `bootstrap/providers.php` - Service provider registration
- `config/fortify.php` - Auth features configuration (2FA, registration, email verification)
- `config/inertia.php` - SSR configuration and page paths

## Auto-Generated Files

The following directories contain auto-generated files and should not be manually edited:

- `resources/js/actions/` - Generated by Wayfinder from controller routes
- `resources/js/routes/` - Generated by Wayfinder from named routes

Regenerate with: `npm run build` or Vite dev server
